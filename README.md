# Anime Website

Website anime modern yang dibangun dengan Next.js, menampilkan informasi anime lengkap dari Jikan API.

## Fitur

- 📱 Responsive Design
- 🎬 Detail Anime Lengkap
- 📺 Daftar Episode
- 🎥 Trailer Anime
- 🎨 Desain Pink & Putih
- ⚡ Fast Loading

## Teknologi

- Next.js 13
- React 18
- Axios
- Jikan API

## Cara Menjalankan

### Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev
